# ✨ Compliment & Roast Generator

A fun, interactive, and beautifully designed web application that generates tailored compliments or edgy roasts based on relationship dynamics and desired tones. Built entirely with Vanilla HTML, CSS, and JavaScript, it features dynamic animated backgrounds, a glassmorphism UI, and built-in webhook integration to connect with external AI workflows.

## 🚀 Features

* **Dual Modes:** Generate sweet, professional, or funny compliments, OR switch gears and generate savage, passive-aggressive, or dark roasts.
* **Dynamic Backgrounds:** The background gradient and floating emojis automatically change based on the selected relationship (Partner, Friend, Colleague, Family, Stranger). 
* **Smart Fallback:** If the webhook fails or is disconnected, the app automatically falls back to a built-in local generation engine so the user never experiences a dead end.
* **Copy to Clipboard:** One-click copying of the generated text with inline success feedback.
* **Lead Magnet Integration:** Includes a "Behind the Magic" modal designed to funnel users toward booking a system consultation.
* **Responsive Design:** "Warm Tech Minimalist" styling with glassmorphism effects, ensuring it looks great on both desktop and mobile devices.

## 🛠 Tech Stack

* **HTML5:** Semantic structure and accessible forms.
* **CSS3:** Custom properties (variables), CSS animations, keyframes, backdrop-filters (glassmorphism), and flexbox.
* **Vanilla JavaScript:** DOM manipulation, asynchronous fetching (webhooks), and randomized string generation. No external libraries or frameworks required.

## ⚙️ Setup & Installation

Since this project uses vanilla web technologies, there is no build step or package installation required.

1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser.
3. The app is immediately ready to use!
